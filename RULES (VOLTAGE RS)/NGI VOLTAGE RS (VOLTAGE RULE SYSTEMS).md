# VOLTAGE SYSTEM (V0.4)

---

## 0. ОПРЕДЕЛЕНИЕ СИСТЕМЫ

VOLTAGE — это модуль правил и безопасности внутри системы NGIX, отвечающий за определение, классификацию и предотвращение нарушений.

Он является структурированным слоем, который:
- определяет, что считается нарушением
- классифицирует поведение пользователей по уровню риска
- выявляет потенциальные угрозы до их развития
- формирует основу для реакций NGIX (суд, санкции, протоколы)

VOLTAGE не управляет системой и не выносит наказания.  
Он формирует сигналы нарушения и уровень опасности для NGIX.

---

# 1. СОЦИАЛЬНЫЕ ЗАПРЕТЫ (SOC)

Запрещено:

- токсичное поведение с целью разрушения общения
- систематические оскорбления участников
- травля (индивидуальная или групповая)
- провоцирование конфликтов между участниками
- разжигание массовых споров и хаоса
- попытки разделить сообщество на враждующие группы
- скрытая агрессия с целью давления на людей
- манипуляция мнением группы участников

---

# 2. ЭКОНОМИЧЕСКИЕ ЗАПРЕТЫ (ECO)

Запрещено:

- создание валюты KSAR или VIR вне системы
- использование багов для получения валюты
- мульти-аккаунты для фарма ресурсов
- циклический фарм наград (повтор действий ради прибыли)
- искусственное увеличение активности ради наград
- передача валюты самому себе через альтернативные аккаунты
- договорные схемы накрутки экономики
- искусственное обесценивание валюты через массовые действия

---

# 3. АДМИНИСТРАТИВНЫЕ ЗАПРЕТЫ (ADM)

Запрещено:

- выдача ролей без оснований системы
- выдача валюты без причины
- использование прав ради личной выгоды
- вмешательство в решения других модераторов без основания
- скрытое управление системой без официального статуса
- принятие решений в конфликте интересов
- изменение правил вне процедур системы

---

# 4. СИСТЕМНЫЕ ЗАПРЕТЫ (SYS)

Запрещено:

- поиск и использование багов системы
- намеренное тестирование уязвимостей без разрешения
- попытки перегрузки сервера (спам, флуд, нагрузка)
- массовое создание действий для нарушения стабильности
- координированные атаки на структуру сервера
- обход правил через технические лазейки
- использование автоматизации (ботов) без разрешения

---

# 5. ОБХОД СИСТЕМЫ (ANTI-EVASION)

Запрещено:

- использование альтернативных аккаунтов для обхода наказаний
- повторный вход после бана без разрешения
- скрытие личности при нарушениях (через смену аккаунтов)
- разбиение одного нарушения на несколько мелких
- попытка обойти правила через переформулировку смысла
- распределённый спам через несколько аккаунтов

---

# 6. СОЦИАЛЬНАЯ ИНЖЕНЕРИЯ

Запрещено:

- выдача себя за администрацию или власть
- ложные заявления о правах или статусе
- выманивание ролей, доступа или валюты
- обман через доверие (ложный авторитет)
- создание фальшивых личностей с целью влияния
- манипуляция участниками через ложную информацию
- давление на админов для получения привилегий

---

# 7. КОНТЕНТНЫЕ ЗАПРЕТЫ

Запрещено:

- незаконный контент
- инструкции по вредоносным действиям
- экстремистские материалы
- угрозы насилия
- пропаганда вредоносного поведения

---

# 8. ОГРАНИЧЕНИЯ 18+

Запрещено:

- размещение сексуализированного контента вне специальных зон
- эротический или порнографический контент в общих каналах
- вовлечение несовершеннолетних в 18+ контент
- обсуждение интимного контента в открытых каналах

---

# 9. МАНИПУЛЯЦИЯ СИСТЕМОЙ

Запрещено:

- попытки сломать экономику
- разрушение структуры ролей
- координированные действия против правил
- создание скрытых групп влияния
- саботаж системы изнутри

---

# 10. СУПЕР-ПРАВИЛО СИСТЕМЫ

Любое действие, которое:

- нарушает стабильность системы
- создаёт массовый хаос
- даёт нечестное преимущество
- обходит структуру правил
- разрушает доверие в Империи

→ считается нарушением, даже если не описано явно

---

# 11. ПРИНЦИП VOLTAGE

Система работает не по буквам правил, а по их смыслу и последствиям.
